## Programming History

<div class="cols">
<div>

* Early languages: **Machine ‚Üí Assembly ‚Üí Structured ‚Üí OOP**
* Machine & Assembly were hardware-dependent.
* Structured Programming (C) introduced functions & flow control.
* Object-Oriented Programming (Java, C++) introduced **classes** & **objects**.
* Java emerged in **1995** by *James Gosling* (Sun Microsystems).

</div>

## What is Java?

<div class="cols">
<div>

* High-level, Object-Oriented, Platform-Independent language.
* Source code compiled to **Bytecode**, executed by JVM.
* Motto: ‚Äú**Write Once, Run Anywhere**.‚Äù
* Designed for portability, reliability, and security.

</div>
<div>
  <div class="imgbox">

![width:800](./assests/java-jvm-pipeline.png)

  </div>
</div>

## JVM (Java Virtual Machine)

<div class="cols">
<div>

* JVM converts bytecode ‚Üí machine instructions.
* Provides abstraction from underlying hardware.
* Components:
  * Class Loader
  * Execution Engine
  * Garbage Collector
* Enables the same code to run across Windows, Linux, and macOS.

</div>

## Java Versions

* **Java 1.0 (1995)** ‚Äì initial release  
* **Java 5 (2004)** ‚Äì generics & annotations  
* **Java 8 (2014)** ‚Äì LTS, introduced lambdas  
* **Java 11 (2018)** ‚Äì LTS, modern baseline  
* **Java 17 (2021)** ‚Äì LTS, sealed classes & records  
* **Java 21 (2023)** ‚Äì LTS, virtual threads  
* **Java 25 (2025)** ‚Äì LTS, pattern matching enhancements
